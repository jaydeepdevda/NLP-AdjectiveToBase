# NLP-AdjectiveToBase
Simple algorithm for convert adjective word to its base form

Rules:

1. remove ed
2. remove y 
3. remove ing
4. remove ness
5. remove able
6. remove ous
7. remove ful

Some Examples:

1. reading -> read 
2. interesting -> interest 
3. talking -> talk 

4. acceptable -> accept
5. creditable -> credit

6. madness -> mad
7. firmness -> firm

8. courageous -> courage
9. dangerous -> danger

10. delightful -> delight
11. forceful -> force
